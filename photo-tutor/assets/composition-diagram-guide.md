# 构图示例图生成指南

## 概览
本文档供智能体参考，用于生成构图示意图和改进对比图。

---

## 示意图类型

### 1. 构图示意图（Composition Diagram）

**用途**：标注照片的构图元素，帮助理解构图原理

**包含元素**：
- 画面边框
- 三分线（虚线）
- 主体位置（●或○）
- 引导线（→或←）
- 框架元素（###或|||）

**生成提示示例**：
```
生成一个构图示意图，标注以下元素：
- 画面三分线（虚线）
- 主体位置在左上角三分线交叉点
- 引导线从左下角指向主体
- 框架元素在画面左侧
```

**输出格式**：简洁的线条图或标注图

---

### 2. 改进前后对比图（Before-After Comparison）

**用途**：对比当前构图和改进构图，直观展示改进效果

**包含元素**：
- 左图：当前构图，标注问题
- 右图：改进构图，标注方案
- 中间：箭头或分隔符

**生成提示示例**：
```
生成一个改进前后对比图：
左图：主体居中，标注"缺少张力"
右图：主体在三分线交叉点，标注"符合三分法"
中间用箭头连接
```

**输出格式**：左右对比，清晰标注

---

### 3. 构图问题标注图（Problem Annotation）

**用途**：标注照片中的构图问题

**包含元素**：
- 原始照片
- 红色标注圈出问题区域
- 文字说明问题

**生成提示示例**：
```
标注这张照片的构图问题：
- 红色圈出主体居中的位置
- 标注"主体居中，缺少张力"
- 标注"缺少引导线"
```

**输出格式**：照片+标注

---

### 4. 构图改进标注图（Improvement Annotation）

**用途**：标注构图改进方案

**包含元素**：
- 改进后的构图示意图
- 绿色标注改进元素
- 文字说明改进方法

**生成提示示例**：
```
标注构图改进方案：
- 绿色标注新的主体位置
- 标注"移动到三分线交叉点"
- 标注"添加引导线"
```

**输出格式**：示意图+标注

```
+-------+-------+

```

## 示意图生成规范

### 颜色规范
- 红色：标注问题
- 绿色：标注改进
- 蓝色：标注引导元素
- 黄色：标注重点

### 线条规范
- 虚线：三分线
- 实线：引导线
- 粗线：框架元素

### 标注规范
- 标注位置：清晰易读
- 标注文字：简洁明了
- 标注大小：适中，不遮挡主体

## 示例生成流程

### 步骤1：识别构图问题
- 分析用户照片
- 参考 [compositional-improvements.md](../references/compositional-improvements.md)
- 确定1-2个主要问题

### 步骤2：选择匹配案例
- 从改进案例中选择最匹配的
- 根据用户照片具体情况调整
- 不要照搬，要定制化

### 步骤3：生成示意图
- 生成当前构图示意图
- 标注问题区域
- 生成改进构图示意图
- 标注改进方案

### 步骤4：提供说明
- 说明为什么这样改进
- 提供实践要点
- 预告预期效果

---

## 注意事项

1. **简洁原则**：示意图应简洁明了，不要太复杂
2. **标注清晰**：标注文字要清晰易读，不要遮挡主体
3. **对比明确**：改进前后的对比要明显
4. **定制化**：根据用户照片的具体情况调整，不要照搬模板
5. **图文结合**：示意图和文字说明要结合，增强理解

## 智能体使用指南

### 何时生成示意图？

- **需要构图改进建议时**：生成改进前后对比图
- **讲解构图理论时**：生成构图示意图
- **标注构图问题**：生成问题标注图

### 如何生成示意图？

1. **分析照片**：识别构图问题
2. **参考案例**：从 [compositional-improvements.md](../references/compositional-improvements.md) 选择匹配案例
3. **生成图像**：根据用户需求生成相应类型的示意图
4. **提供说明**：配合文字说明，图文结合

## 示例输出格式

### 格式1：示意图+说明

```
## 构图问题：主体居中

当前构图：
[示意图]
标注：主体位于画面正中央，缺乏视觉张力

改进方案：
[示意图]
标注：将主体移动到三分线交叉点

实践要点：
1. 向右移动拍摄位置
2. 保持主体与相机的距离
3. 确保背景简洁

预期效果：画面更有动感，视觉焦点更突出
```

### 格式2：对比图+详细说明

```
## 构图改进方案

改进前后对比：
[左图：主体居中] → [右图：三分法构图]

问题分析：
- 当前构图：主体居中，画面平淡
- 缺少引导线，视线无目标

改进方法：
- 将主体移动到右上角三分线交叉点
- 利用场景中的栏杆作为引导线
- 简化背景，突出主体

详细步骤：
1. 观察场景，找到合适的三分线位置
2. 调整拍摄角度，让主体落在交叉点
3. 寻找引导线元素（栏杆、道路等）
4. 调整焦距，简化背景

预期效果：构图更加平衡，视觉引导更清晰
```
